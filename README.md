# "Wackerspace"

The website for "Wackerspace", the Wellington Hackerspace.

<https://wackerspace.com>

## Development

### Install

With `git`, `node`, and `pnpm` installed:

```shell
git clone git@github.com:wackerspace:wackerspace.com
cd wackerspace.com
pnpm install
```

### Run dev server

```shell
pnpm run dev
```


### Lint & format

```shell
pnpm run lint
```

```shell
pnpm run format
```

```shell
pnpm run format:unsafe
```
